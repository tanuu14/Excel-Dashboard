# Excel Coffee Sales Dashboard

A dynamic and interactive sales dashboard built in Microsoft Excel using pivot tables, slicers, and charts to analyze monthly performance.

## Project Overview

This project showcases my ability to:
- Clean and organize raw sales data
- Create insightful pivot tables and visuals
- Build a user-friendly dashboard interface
- Derive actionable insights from sales trends

## Tools Used

- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning Techniques

## Key Insights

- Identified top-performing regions and top customers 
- Analyzed monthly revenue trends and growth patterns
- Tracked sales rep performance over time

## How to Use

Download the Excel file from this repo and open it in Excel. Use the slicers to interact with the dashboard and filter results by type, size or loyalty card.
